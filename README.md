# 🎬 Movie Explorer

A modern and responsive movie discovery platform built with **React**, **TMDB API**, **Bootstrap 5**, and **Framer Motion**. The application enables users to explore trending movies, perform real-time searches, filter and sort results, manage favorites, and view detailed movie information through an interactive modal interface.

Designed with a focus on performance, user experience, responsive design, and component-based architecture.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/namanjainb3-tech/movie-explorer.git
```

### Navigate to the Project

```bash
cd movie-explorer
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
