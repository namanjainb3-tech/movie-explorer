# 🎬 Movie Explorer

A modern and responsive movie browsing web application built using React and TMDB API. Users can explore popular movies, search in real-time, apply filters, and save their favorite movies.

---

## 🚀 Live Demo

🔗 https://movie-explorer-pink-seven.vercel.app/

---

## ✨ Features

* 🔍 Search movies by title
* ⭐ Filter movies by rating
* 📊 Sort by popularity, rating, or title
* ❤️ Add/remove favorites (saved in localStorage)
* 🌙 Dark mode toggle
* 🎥 Click on a movie to view detailed information (modal)
* 📱 Fully responsive design
* ⚡ Smooth animations using Framer Motion

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Bootstrap 5
* **Animations:** Framer Motion
* **API:** TMDB (The Movie Database)
* **Deployment:** Vercel

---

## 🧠 Architecture

The project follows a **component-based architecture** for better scalability and maintainability.

```
src/
│── App.jsx
│── Navbar.jsx
│── MovieCard.jsx
│── Modal.jsx
│── Controls.jsx
```

* `Navbar` → Search + Dark Mode
* `MovieCard` → Movie UI + Favorites
* `Modal` → Movie details popup
* `Controls` → Filter + Sort

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/namanjainb3-tech/movie-explorer.git
```

2. Navigate to project:

```
cd movie-explorer
```

3. Install dependencies:

```
npm install
```

4. Run locally:

```
npm run dev
```
